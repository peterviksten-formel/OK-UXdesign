# The Opportunity Map

Every touchpoint on a typical web experience where personality can be expressed — most of which are treated as functional obligations rather than brand moments. Use this as a checklist when diagnosing a page or flow, not as a menu to hand to the user.

## The full table

| Surface | Default treatment | Passion treatment |
|---|---|---|
| **Page title / browser tab** | "[Product] — [Tagline]" | State-aware ("12 unread"), seasonal, animated, or narrative-changing |
| **Loading screen** | Spinner or skeleton | Branded illustration, progress narrative, humor that acknowledges the wait, countdown, music |
| **Error messages** | "Something went wrong" / "An error occurred" | Empathetic, specific, actionable, in brand voice, ideally naming what *actually* happened |
| **404 page** | "Page not found" + link to home | Easter egg, brand story, useful redirect with character, an editorial moment in its own right |
| **Empty states** | Blank canvas | Guidance, invitation, starter content, illustration, personality — an opportunity to teach and charm simultaneously |
| **Form field labels** | "Email", "Name" | Conversational, context-aware ("Your best email address", "What should we call you?") — voice-appropriate |
| **Form field placeholders** | "example@example.com" | Illustrative in brand voice ("e.g., hola@greatcompany.com"), sometimes playful |
| **Success confirmations** | "Done!" / "Submitted." | Celebration, next-step guidance, warmth (Mailchimp's Freddie high-five) |
| **Button labels** | "Submit", "Click here" | Specific to the outcome, sometimes in-voice ("Let's go", "Send it", "Plant the flag") |
| **Hover states** | Color change | Character-revealing motion, micro-animation, unexpected detail that rewards pointing |
| **Scroll behavior** | Sections appear | Narrative unfolds, reveal animations with editorial intent, sense of progression |
| **Footer** | Legal + nav links | Brand personality, unexpected copy, editorial moment, conversation with the user |
| **Cursor** | Default arrow | Brand-expressive custom cursor, trail, contextual state change |
| **Navigation** | Dropdown menus | Hover reveals with imagery, editorial teases, micro-animations that set tone |
| **About page** | Team bios + history | Voice-driven storytelling, honest chaos, process glimpses, a window into who actually made this |
| **Search** | Results list | Personality in "no results" states, unexpected suggestions, tone in search hints |
| **Page transitions** | Cut / fade | Brand-expressive motion that carries identity between states |
| **Onboarding empty states** | Blank or "Get started" | Starter content, illustrated guidance, tone that sets expectations for the product |
| **Toasts / notifications** | "Saved" | In-voice confirmation that positions the brand at a moment of relief or celebration |
| **Email receipts / transactional email** | Template | Voice extends into the inbox — where most brands disappear |
| **Terms of service / privacy page** | Legalese | Plain-language version alongside the legal text; or legal text itself written with care (see Basecamp) |
| **Maintenance / offline pages** | "We'll be back soon" | Honest explanation of what's happening and why; character under pressure |

## How to use this map

**Don't treat it as a checklist to exhaust.** A page that attempts personality at every surface becomes noise — the user loses the ability to feel any single moment. The goal is to identify the 3-5 surfaces where the gap between the default and a thoughtful treatment is widest *and* where the user's attention is highest *and* where the emotional charge of the moment is highest.

## The three ranking questions

For each surface, ask:

1. **Attention** — is the user actually here? A 404 page in an app where 404s happen twice a year gets almost no eyeballs. An empty state on the core product's main surface gets millions.
2. **Emotional charge** — is the user feeling something meaningful at this moment? (Waiting, confused, relieved, anxious, curious, delighted.) Emotionally charged moments are where visible investment pays outsized returns.
3. **Distance from default** — how far is what's currently there from what a brand that genuinely cared would do? The wider the gap, the larger the opportunity.

Multiply these together in your head. The highest-scoring surfaces are where you focus.

## A note on "invisible" surfaces

Some surfaces are rarely consciously noticed but felt everywhere: the hover state on a secondary nav link, the typographic rhythm of documentation, the quality of a dashboard chart's empty state. Stripe's lesson is that applying the same care to these invisible surfaces is what separates "designed" from "engineered-to-feel-inevitable." These won't usually be in the user's top 3 conscious touchpoints, but a brand that wants to earn love at the systemic level will invest here anyway.

When the user is at a level of maturity where the obvious surfaces are already handled, point them at these. Otherwise, start with the visible ones.
